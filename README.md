# 📊 Auditoría de Productividad Digital
Repositorio automatizado para el seguimiento de entregas en Google Drive.

## 🚀 Acceso al Dashboard
El reporte interactivo está disponible en:
**[Ver Dashboard en GitHub Pages](https://github.com/tu-usuario/tu-repo)**

## ⚙️ Funcionamiento
1. `auditoria.py` escanea recursivamente la carpeta de Drive.
2. Sincroniza los datos con Google Sheets.
3. Genera un `index.html` estático con los resultados consolidados.
